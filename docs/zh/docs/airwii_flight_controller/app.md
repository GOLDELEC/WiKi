# 遥控App使用说明

AirWii飞行器集成了能跟智能手机直接通信的WiFi模块，这样通过专用的手机软件App就能控制AirWii飞行器的飞行。

遥控AirWii飞行器的App能够完全模拟物理遥控器的功能，而且用户能够在视线完全脱离手机屏幕的情况下，准确操作摇杆，达到自由控制飞行的效果。
